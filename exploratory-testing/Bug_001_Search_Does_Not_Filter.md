# Bug 001 - Search Does Not Filter Projects by Partial Input

## Environment
- OS: Windows 11  
- Browser: Microsoft Edge  
- Application: RMSys  

## Preconditions
- User logged in  
- Project tab open
- Default filter set to “Show Active”  

## Steps to Reproduce
1. Go to Project tab  
2. Enter "D" in the search bar  
3. Click search  

## Actual Result
All projects are displayed, including projects that do NOT contain the letter “D” in their name

## Expected Result
Only projects containing the letter “D” in their name should be displayed
